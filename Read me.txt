## Synopsis
This project consists of a drop down menu.


## Installation
In order to dowload the project folder follow these steps:


Step 1: Click "Download Zip File" or "Get File" button to get files.


Step 2: DO NOT OPEN THE FOLDER AND TRUN THE PROJECT IMMEDIEATLY. as this will show a broken website. Frist exract all files. 
Once all files are extracted, than you can run the index.html file or browse all files.

Step 3: Set up your web server and move extracted file into your web server.

Step 4: Run website
